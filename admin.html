📋 How to Fulfill POD Orders (Printify Manual Flow)

1
Customer places order → you receive notification (check Orders tab)
2
Verify payment: check bank account for transfer with Order ID in note
3
Go to printify.com → Create manual order → Select product + variant (size/color)
4
Enter customer shipping address → Place order on Printify
5
Mark order as Confirmed in Orders tab → email tracking to customer
6
When shipped: mark Shipped + add tracking number in notes

📊 Google Sheet Integration
📋 How to connect Google Sheet:
Step 1: Create a Google Sheet with these columns:
id, name, sku, type, price, original_price, image_url, description, tags, status, shipping, emoji

Step 2: File → Share → Publish to web → Select "Sheet1" → CSV format → Copy link

Step 3: Paste the link in shop.html line: const SHEET_CSV_URL = 'PASTE_HERE';

Orders Sheet: Create "Sheet2" named "Orders" — orders will auto-append via webhook (optional setup).
