Bước 1 — Tạo tab Orders trong Google Sheet
Mở Google Sheet của bạn (cùng sheet với sản phẩm) → click dấu + dưới cùng → đổi tên thành Orders
Thêm header ở hàng 1: Timestamp | OrderRef | Name | Email | Phone | Items | Total | Note | Status
Bước 2 — Tạo Apps Script
Trong Google Sheet → menu Extensions → Apps Script → xoá code cũ → paste đoạn sau:
function doPost(e) { try { const sheet = SpreadsheetApp .getActiveSpreadsheet() .getSheetByName('Orders'); const data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date().toLocaleString('vi-VN'), data.order_ref || '', data.name || '', data.email || '', data.phone || '', data.items || '', data.total || '', data.note || '', data.status || 'Pending' ]); return ContentService .createTextOutput(JSON.stringify({result:'ok'})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({error:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
Bước 3 — Deploy
Nhấn Deploy → New deployment → chọn type Web app
• Execute as: Me
• Who has access: Anyone
→ Nhấn Deploy → Copy URL dài xuất hiện
Bước 4 — Paste URL vào đây
Paste URL vừa copy vào ô bên trên → nhấn Save → nhấn Test để kiểm tra.
⚠️ Sau khi deploy lần đầu, nếu Google hỏi quyền → nhấn Review permissions → Allow
